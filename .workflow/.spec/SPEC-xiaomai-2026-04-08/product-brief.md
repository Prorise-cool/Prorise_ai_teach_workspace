# Product Brief: 小麦

> 本文档复用现有规划文档，完整内容请查看源文件。

## Vision

**中国首个 AIGC 原生虚拟教室** —— 多模态认知智能体驱动的沉浸式个性化教学空间，重新定义数字化教育范式。

## Core Value

**一键生成教学视频与课堂** —— 用户只需输入题目或主题，系统自动完成理解、分镜、渲染、合成全链路。

## Target Users

| 用户群体 | 核心需求 | 使用场景 |
|----------|----------|----------|
| 高职学生 | 高效理解抽象概念、个性化学习 | 课后复习、考前冲刺、作业难题 |
| 高职教师 | 快速制作高质量教学视频 | 备课、翻转课堂、教学资源建设 |
| 自学者 | 按需学习、补足知识盲点 | 技能提升、职业转型 |

## Core Capabilities

### Layer A: 双内容引擎（独立）
- **Video Engine**: 题目输入 → 讲解视频生成
- **Classroom Engine**: 主题输入 → 课堂内容生成

### Layer B: 会话伴学 Companion（共享）
- 围绕当前学习上下文进行即时追问与解释
- 绑定视频时间点、课堂步骤或白板步骤

### Layer C: Evidence / Retrieval Service
- 教材、讲义、术语与上传文档的可检索证据层
- 为 Companion 与 Learning Coach 提供来源引用

### Layer D: Learning Coach
- 课后 checkpoint / quiz、错题解析、推荐、学习路径规划

### Layer E: 结果回看与数据闭环
- 历史记录、收藏管理、会话问答回写、学习信号沉淀

## Success Metrics

| 指标类型 | 目标值 |
|----------|--------|
| 生成成功率 | > 80% |
| 端到端延迟 | < 5 分钟 |
| 学生满意度 | > 4.0/5.0 |
| 教师备课效率提升 | 50% |
