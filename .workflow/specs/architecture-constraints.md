# Architecture Constraints

## System Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                    Nginx 统一入口                            │
└──────────────────────┬──────────────────────────────────────┘
                       │
         ┌─────────────┼─────────────┐
         ▼             ▼             ▼
┌─────────────┐ ┌─────────────┐ ┌─────────────┐
│ React SPA   │ │ FastAPI     │ │ RuoYi       │
│ :5173       │ │ :8090       │ │ :8080       │
│ (ToC 前端)   │ │ (功能服务)   │ │ (持久化)     │
└─────────────┘ └──────┬──────┘ └──────┬──────┘
                       │                │
                ┌──────┴──────┐         │
                ▼             ▼         ▼
         ┌──────────┐  ┌──────────┐ ┌──────────┐
         │ Redis    │  │ Worker   │ │ MySQL    │
         │ (运行态)  │  │ (渲染)    │ │ (持久化)  │
         └──────────┘  └──────────┘ └──────────┘
```

## Responsibility Boundaries

### FastAPI (功能服务)
- 课堂生成、视频生成
- Companion、Evidence、Learning Coach
- Provider 调度、SSE、异步任务协调
- **禁止**：长期业务数据主存储

### RuoYi (持久化)
- 用户、角色、菜单、审计日志
- 小麦业务表、会话记录、学习数据
- CRUD 管理与权限控制

### Redis (运行态)
- Token 在线态
- 任务状态
- 事件缓存
- 会话临时上下文
- **禁止**：长期业务数据

## Data Flow Rules

1. 长期业务数据 → RuoYi/MySQL 或 COS
2. 运行态状态 → Redis
3. 前端 ↔ FastAPI ↔ RuoYi（防腐层）

## Engine Independence

- Video Engine 与 Classroom Engine 生成链路保持独立
- 共享基础设施，不合并流水线
- 统一的是会话交互语义层（Companion），不是内容生成引擎
