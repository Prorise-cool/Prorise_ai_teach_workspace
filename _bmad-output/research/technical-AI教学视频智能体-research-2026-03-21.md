## 二、单机部署优化

| 问题 | 优化方案 |
|------|----------|
| FastAPI 验证权限要调 RuoYi HTTP | **共享 Redis**，JWT Token 两边验证，无需 HTTP |
| 用户数据要同步 | **共享 MySQL**，FastAPI 只读用户表 |
| 前端入口分散 | **Nginx 统一入口**，按路径分发 |

## 三、复用策略

### OpenMAIC（`references/openmaic/`）

**复用：** 全部后端能力，前端重做
- LangGraph 多 Agent 编排
- 课堂生成流水线
- 动作执行引擎
- LLM/TTS 服务商抽象

**许可证：** AGPL-3.0 ✅

### ManimToVideoClaw（`references/manim-to-video-claw/`）

**复用：** 视频生成管线，合并入 FastAPI
- 分镜生成
- Manim 代码生成 + 自修复
- 渲染 + TTS + 合成

**许可证：** 商业软件（自有）✅
