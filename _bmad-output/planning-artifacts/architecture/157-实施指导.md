# 15.7 实施指导

## AI Agent 指导原则

1. **严格遵循所有架构决策** - 不偏离本文档定义的决策
2. **使用定义的实施模式** - 命名、结构、通信模式
3. **尊重项目结构和边界** - 不跨模块引入依赖
4. **遇到问题优先查阅本文档** - 搜索关键词定位规则
