# 9. Quiz Interaction Design（课后小测交互设计）

> **架构对齐**：本功能通过 FastAPI 调用腾讯云智能体平台的工作流能力，生成测验题目。

## 9.1 功能定位

**触发场景**：课堂结束后自动生成 / 用户主动请求

**功能来源**：学习辅助模块（🟢 腾讯云工作流）

## 9.2 交互流程

```
课堂结束
    ↓
[弹窗] "课堂学完了，来做个小测验吧！"
    ↓
[用户点击开始]
    ↓
┌─────────────────────────────────────┐
│  📝 第 1 题 / 共 5 题                 │
│                                     │
│  函数 f(x)=x²-2x+1 的极小值是？      │
│                                     │
│  ○ A. 0                             │
│  ○ B. 1                             │
│  ○ C. -1                            │
│  ○ D. 2                             │
│                                     │
│              [提交答案]              │
└─────────────────────────────────────┘
    ↓
[用户选择答案]
    ↓
┌─────────────────────────────────────┐
│  ✅ 正确！                           │
│                                     │
│  解析：                              │
│  对 f(x) 求导得 f'(x)=2x-2           │
│  令 f'(x)=0，得 x=1                  │
│  f(1)=1-2+1=0                        │
│                                     │
│         [下一题 →]                   │
└─────────────────────────────────────┘
    ↓
... 重复直到完成所有题目 ...
    ↓
┌─────────────────────────────────────┐
│  🎉 测验完成！                       │
│                                     │
│  正确率：4/5 (80%)                   │
│  用时：2 分 30 秒                    │
│                                     │
│  📊 知识点掌握情况：                  │
│  ✅ 极值求解                         │
│  ✅ 导数计算                         │
│  ⚠️ 二阶导判断                       │
│                                     │
│  [查看错题] [再来一次] [返回课堂]     │
└─────────────────────────────────────┘
```

## 9.3 交互规范

| 阶段 | 元素 | 交互 | 反馈 |
|------|------|------|------|
| **答题** | 单选按钮 | 点击选择 | 选中高亮 |
| **答题** | 提交按钮 | 点击 | 禁用选项 + 显示结果 |
| **结果** | 正确/错误 | 自动显示 | ✅ 绿色 / ❌ 红色 |
| **解析** | 展开区域 | 自动展开 | 显示详细解析 |
| **完成** | 结果卡片 | 自动弹出 | 显示统计数据 |
