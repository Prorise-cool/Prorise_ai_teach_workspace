# 8. Knowledge QA Page（知识问答页）

> **架构对齐**：本页面通过 FastAPI 调用腾讯云智能体平台的知识库 RAG 能力，前端 UI 100% 自研。

## 8.1 页面定位

**路由**：`/knowledge`

**功能来源**：课程知识库模块（🟢 腾讯云 API）

**核心能力**：
- 知识库问答（RAG 检索）
- 引用来源展示
- 知识点关联推荐

## 8.3 交互规范

| 元素 | 交互 | 反馈 |
|------|------|------|
| **课程选择器** | 点击切换 | 下拉菜单 + 当前选中高亮 |
| **问题输入** | 输入 + Enter | 发送问题 + 显示加载状态 |
| **AI 回复** | 流式显示 | 逐字显示，模拟打字效果 |
| **引用来源** | 点击 | 跳转到原文位置（如可用） |
| **相关知识点** | 点击 | 自动填充到输入框 |

## 8.4 数据流

```
用户输入问题
    ↓
前端 → FastAPI /api/v1/knowledge/chat
    ↓
FastAPI → TencentADPKnowledgeProvider
    ↓
腾讯云智能体平台 → RAG 检索
    ↓
返回：回复内容 + 引用来源 + 相关知识点
    ↓
前端渲染回复
    ↓
后台异步：问答记录回写 RuoYi xm_knowledge_chat_log
```

## 8.5 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/knowledge/chat` | POST | 发送问答请求 |
| `/api/v1/knowledge/courses` | GET | 获取课程列表 |
| `/api/v1/knowledge/history` | GET | 获取问答历史 |
