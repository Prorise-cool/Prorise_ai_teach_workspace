# 10. Learning Path Design（学习路径规划）

> **架构对齐**：本功能通过 FastAPI 调用腾讯云智能体平台的 Multi-Agent 能力。

## 10.1 功能定位

**触发场景**：用户主动请求 / 系统根据学习记录推荐

**功能来源**：学习辅助模块（🟢 腾讯云 Multi-Agent）

## 10.2 交互流程

```
用户点击 "规划学习路径"
    ↓
[弹窗] 选择学习目标
    ↓
┌─────────────────────────────────────┐

│  📅 目标完成时间：                    │
│  [4 周 ▼]                            │
│                                     │
│         [开始规划]                   │
└─────────────────────────────────────┘
    ↓
[生成中...]
    ↓
```

## 10.3 交互规范

| 元素 | 交互 | 反馈 |
|------|------|------|
| **学习节点** | 点击 | 展开/收起详细内容 |
| **完成状态** | 点击完成 | 打勾 + 进度更新 |
| **调整计划** | 点击 | 进入编辑模式 |
| **开始学习** | 点击 | 跳转到对应内容 |
