# 16. 附录：与旧 PRD 的继承说明

## 16.1 继承内容

本版 PRD 继承并重构了旧版中仍然有效的内容：

* 双入口产品愿景
* 高职教育垂直场景定位
* 高职学生 / 教师 / 自学者等目标用户定义
* 用户旅程中的主题学习、单题讲解、教师备课核心场景
* 视频成功率、课堂成功率、满意度、效率提升等成功指标
* MVP 哲学、测试门禁、变更控制方法

## 16.2 主要升级点

本版 PRD 相较旧版的关键升级包括：

1. 将 Evidence / Retrieval 服务正式纳入产品主干
2. 将 Learning Coach 学习教练服务正式纳入产品主干
3. 将学习中心聚合页纳入正式页面范围
4. 将统一任务框架、SSE 恢复、Provider Failover 产品化表达
5. 明确 Async Worker、RuoYi 落表、腾讯云 ADP 的边界关系
6. 将“只能演示生成结果”升级为“结果可回看、可聚合、可追溯”的闭环能力
