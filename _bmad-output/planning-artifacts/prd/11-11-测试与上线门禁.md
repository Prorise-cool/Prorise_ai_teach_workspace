# 11. 测试与上线门禁

## 11.1 E2E 必过场景

### P0 必过

1. 用户登录 → 进入首页 → 进入单题视频场景
2. 文本题目输入 → 视频生成 → 播放视频
3. 图片上传 → OCR → 视频生成 → 播放视频
4. 主题课堂输入主题 → 生成课堂 → 查看幻灯片 → 发起 Companion 追问 → 进入 checkpoint / quiz
5. 会话结果页或学习中心触发“查看依据” → 打开来源抽屉 / 证据面板 → 查看状态提示
6. Learning Coach 生成基础 checkpoint / quiz → 提交答案 → 查看结果
7. 长任务过程中进度展示正常，连接中断后状态可恢复
8. 任务完成后可在历史记录或学习中心重新打开结果
9. 登录页、首页、输入页、结果页与学习中心域关键静态文案通过 i18n 资源正确渲染，默认中文不出现散落硬编码漏网项
10. 登录页、首页、输入页、结果页与学习中心域关键交互在亮色 / 暗色模式下保持可读、可用、不破版

### P1 必过

1. 个人中心查看基础资料
2. 历史记录展示与跳转
3. 风格切换对生成结果有可辨差异
4. 学习路径建议在学习中心可查看
5. 推荐内容可跳转相关学习结果

### P2 可后置

1. 收藏夹增强管理
2. 中英文切换
3. 多 Agent 深度讨论
4. 白板高级布局

## 11.2 上线门禁

以下条件不满足时，不允许发版：

* 所有 P0 E2E 场景 100% 通过
* P0 Bug = 0
* Lint / Type / 测试通过
* 关键性能指标达到基线
* 任务状态、进度、完成、失败四类路径均验证通过
* 长期业务数据落表与文件归档路径验证通过
* 关键页面默认语言资源与亮色 / 暗色模式检查通过
