# 6.2 FR-UI 前端页面与导航

## FR-UI-001 首页双入口展示
**描述**  
首页需清晰展示“初学场景”和“问答场景”两个核心入口，并帮助用户理解差异。

**优先级**  
P0

**验收标准**
- AC1：首页展示两个核心入口
- AC2：每个入口包含简明文案说明适用场景
- AC3：用户点击入口后可在 1 秒内进入对应功能页面

## FR-UI-002 课堂页面
**描述**  
课堂页面承载课堂内容展示、幻灯片浏览、用户交互和测验结果反馈。

**优先级**  
P0

**验收标准**
- AC1：页面包含幻灯片区域、讲解区域和用户输入区域
- AC2：用户可顺序切换浏览幻灯片
- AC3：课堂完成后可进入测验与结果反馈

## FR-UI-003 视频生成页
**描述**  
视频生成页承载题目输入、图片上传、风格选择、任务创建与进度查看。

**优先级**  
P0

**验收标准**
- AC1：页面提供文本输入框与图片上传区域
- AC2：页面支持风格选择或默认风格
- AC3：任务提交后展示当前生成进度与阶段提示

## FR-UI-004 播放器页
**描述**  
播放器页承载视频播放、倍速、全屏、信息查看及结果操作。

**优先级**  
P0

**验收标准**
- AC1：播放器位于页面主体区域
- AC2：播放器下方显示题目信息或知识点标签
- AC3：提供收藏、重新生成、返回等基础操作入口

## FR-UI-005 个人中心
**描述**  
个人中心展示用户基础资料、历史记录、收藏、设置等信息。

**优先级**  
P1

**验收标准**
- AC1：页面展示用户头像、昵称、学校信息
- AC2：页面提供历史记录与设置入口
- AC3：页面支持个人信息修改入口

## FR-UI-006 知识问答页
**描述**  
知识问答页承载知识库问答、术语解释、引用来源与文档解析后的问答体验。

**优先级**  
P0

**验收标准**
- AC1：页面支持用户输入知识问题并发起提问
- AC2：系统返回回答时可展示引用来源或文档片段
- AC3：回答失败时用户获得明确提示并可重试

## FR-UI-007 学习中心页
**描述**  
学习中心用于聚合学习记录、收藏、问答记录、测验结果与推荐内容。

**优先级**  
P0

**验收标准**
- AC1：页面聚合展示至少历史记录、收藏、测验结果三个区域
- AC2：用户可从学习中心进入对应明细结果页
- AC3：页面展示结构清晰，不因聚合导致信息混乱

## FR-UI-008 国际化架构预留
**描述**  
前端需为中英双语切换保留架构能力。

**优先级**  
P2

**验收标准**
- AC1：静态文本具备国际化资源管理方式
- AC2：未来接入语言切换时不需要重写页面结构
- AC3：当前版本默认中文，不阻断 MVP 交付
