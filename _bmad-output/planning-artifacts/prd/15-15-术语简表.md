# 15. 术语简表

| 术语 | 说明 |
|---|---|
| 课堂服务 | 入口 1 对应的基础课堂生成能力 |
| 视频服务 | 入口 2 对应的单题讲解视频生成能力 |
| Evidence / Retrieval | 基于知识库与外部能力的课程证据补充与来源引用能力，当前版本只通过非路由证据面板承载 |
| Learning Coach | checkpoint / quiz、错题解析、学习路径、推荐等学后巩固能力 |
| FR-KQ（历史编号） | 历史编号保留，不代表学生端存在独立 Knowledge QA 页面；当前语义统一对应 Evidence / Retrieval 服务层 |
| Agent 风格 | 老师人格预设，不是按老师切换的页面主题；页面仅存在统一品牌体系下的亮 / 暗色基础主题 |
| Provider | 外部能力抽象，如 LLM / TTS / OCR / ADP |
| Failover | 主 Provider 失败后切换备选 Provider |
| 长期业务数据 | 需要持久化、查询、审计的数据 |
| 运行时状态 | 短期任务状态、SSE 事件、会话上下文 |
| Async Worker | 用于执行长耗时任务的独立执行进程 |

## 15.1 术语映射（冻结）

| 最终业务术语 | 页面表现 | 技术实现名 | 历史旧名 |
|---|---|---|---|
| Video Engine | 视频输入 / 等待 / 播放页 | `features/video` | 视频服务 |
| Classroom Engine | 课堂输入 / 等待 / 结果页 | `features/classroom` | 课堂服务 |
| Companion | 伴学侧栏 / 白板解释 | `features/companion` | 会话伴学 |
| Evidence / Retrieval | 来源抽屉 / 证据面板 | `features/knowledge` | Knowledge / 知识问答 |
| Learning Coach | checkpoint / quiz / path | `features/learning` | 学习辅助 |
