# 11.1 E2E 必过场景
## P0 必过
1. 用户登录 → 进入首页 → 进入问答场景
2. 文本题目输入 → 视频生成 → 播放视频
3. 图片上传 → OCR → 视频生成 → 播放视频
4. 初学场景输入主题 → 生成课堂 → 查看幻灯片 → 完成测验
5. 知识问答页输入问题 → 返回答案 → 查看来源或状态提示
6. 学习辅助生成基础小测 → 提交答案 → 查看结果
7. 长任务过程中进度展示正常，连接中断后状态可恢复
8. 任务完成后可在历史记录或学习中心重新打开结果

## P1 必过
1. 个人中心查看基础资料
2. 历史记录展示与跳转
3. 风格切换对生成结果有可辨差异
4. 学习路径建议在学习中心可查看
5. 推荐内容可跳转相关学习结果

## P2 可后置
1. 收藏夹增强管理
2. 中英文切换
3. 多 Agent 深度讨论
4. 白板高级布局
