# 9. 成功标准

## 9.1 产品成功指标

| 指标 | 目标值 | 说明 |
|---|---|---|
| 视频生成成功率 | > 80% | 成功返回可播放视频 |
| 基础课堂生成成功率 | > 90% | 课堂链路完成且前端可展示 |
| 视频生成端到端延迟 | P95 < 5 分钟 | 从提交到可播放结果 |
| 学生满意度 | > 4.0 / 5.0 | 基于生成任务后的评分 |
| 教师效率提升 | > 50% | 相比传统制作方式的平均耗时下降 |
| 7 日留存率 | > 30% | 用于验证学习闭环价值 |

## 9.2 技术成功标准

* 长任务必须具备任务状态与进度反馈
* 视频、课堂、证据检索、Learning Coach 结果均可回看
* 任务失败时系统返回明确失败结果，不静默丢失
* 长期业务数据有明确归属
* 关键链路支持日志追踪与问题排查
* Provider 切换与降级路径可验证
