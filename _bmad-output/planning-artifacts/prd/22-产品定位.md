# 2.2 产品定位
小麦当前提供三类核心能力层：

## 核心能力层 A：双入口学习体验
- **入口 1：初学场景**
  - 用户输入主题
  - 系统生成基础课堂内容、幻灯片、课堂测验与互动讲解
- **入口 2：问答场景**
  - 用户输入题目文本或上传题目图片
  - 系统生成单题讲解视频并支持播放复习

## 核心能力层 B：知识与学习辅助
- 课程知识库问答
- 术语解释
- 文档解析后知识问答
- 错题解析
- 课后小测
- 学习路径规划
- 知识点推荐
- 错题本与学习中心聚合

## 核心能力层 C：结果回看与数据闭环
- 历史记录
- 收藏管理
- 测验结果记录
- 问答记录回写
- 学习中心聚合展示
