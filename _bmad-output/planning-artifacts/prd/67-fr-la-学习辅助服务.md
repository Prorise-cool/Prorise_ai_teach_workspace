# 6.7 FR-LA 学习辅助服务

## FR-LA-001 课后小测生成
**描述**  
系统可基于课堂、知识问答或知识点生成课后小测。

**优先级**  
P0

**验收标准**
- AC1：系统可生成基础小测题目
- AC2：用户提交后可看到结果与解析
- AC3：测验结果进入学习记录与学习中心

## FR-LA-002 错题解析
**描述**  
系统可针对用户错题返回解析内容。

**优先级**  
P0

**验收标准**
- AC1：系统可识别错题并生成对应解析
- AC2：解析结果可帮助用户理解错误原因
- AC3：错题解析结果可再次查看

## FR-LA-003 学习路径规划
**描述**  
系统可根据用户当前学习主题、测验结果或问答记录生成基础学习路径建议。

**优先级**  
P1

**验收标准**
- AC1：系统可生成分步骤学习建议
- AC2：路径结果与当前知识主题相关
- AC3：用户可在学习中心查看路径建议

## FR-LA-004 知识点推荐
**描述**  
系统可根据用户当前学习内容推荐后续知识点。

**优先级**  
P1

**验收标准**
- AC1：推荐结果与用户最近学习内容具有关联性
- AC2：推荐条目数量控制在可理解范围内
- AC3：用户可从推荐结果跳转到相关学习内容

## FR-LA-005 错题本
**描述**  
系统可将错题结果归档为错题本，供用户后续复习。

**优先级**  
P1

**验收标准**
- AC1：错题可进入错题本列表
- AC2：用户可再次查看题目、答案与解析
- AC3：错题本作为长期数据持久化保存

## FR-LA-006 学习中心聚合展示
**描述**  
学习中心可聚合展示课后小测、错题解析、学习路径、知识推荐等结果。

**优先级**  
P0

**验收标准**
- AC1：用户可在学习中心查看多个学习辅助结果入口
- AC2：各项结果可重新打开对应详情
- AC3：页面结构清晰，不混淆不同类型内容
