## 快速开始

### 前置要求

列出使用该项目所需的前置条件及安装方法：

* Node.js 18+
* npm 或 yarn

### 安装步骤

1. 克隆仓库
   ```sh
   git clone https://github.com/Prorise-cool/Prorise_ai_teach_workspace.git
   ```
2. 安装依赖
   ```sh
   npm install
   ```
3. 启动开发服务器
   ```sh
   npm run dev
   ```
