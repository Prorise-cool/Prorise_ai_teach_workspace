# 贡献指南

感谢您考虑为 Prorise_ai_teach_workspace 做出贡献！

## 如何贡献

### 报告 Bug

如果您发现了 Bug，请打开一个 Issue 并使用 Bug 报告模板。请务必包含：
- 清晰的 Bug 描述
- 复现步骤
- 预期和实际行为
- 环境信息

### 提交 Pull Request

1. Fork 本仓库
2. 从 `main` 分支创建一个新分支 (`git checkout -b feature/your-feature`)
3. 进行更改
4. 确保所有测试通过
5. 提交您的更改并推送
6. 在 GitHub 上创建 Pull Request

### 代码规范

- 使用有意义的变量和函数命名
- 添加必要的注释
- 确保代码通过 lint 检查
- 为新功能添加测试

## 工作流程

1. 创建一个描述您想要解决的问题的 Issue
2. 在 Issue 中讨论您的方案
3. 获得批准后，实现您的更改
4. 提交 PR 并等待审核
