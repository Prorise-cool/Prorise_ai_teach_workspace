# RuoYi 代码生成器代码存储

本目录用于存放 RuoYi 后台代码生成器导出的规范产物。

当前约定：

- 每张表独立一个目录。
- 每张表目录下固定保留 `backend`、`frontend`、`sql` 3 个子目录。
- `backend`：存放 Java 后端生成代码。
- `frontend`：存放 Soybean / Vue 前端生成代码。
- `sql`：存放代码生成器导出的菜单 SQL 或后续补充 SQL。
