## 3. 视觉构思建议 (Inspiration Board)

### A. 麦穗 + AI 节点
- **方案**：将麦穗的一粒粒麦子设计成圆点（Nodes），并用极细线条连接，形成类似神经网络的结构。
- **推荐色**：#f5c547 (金) + #3b1701 (深棕文字)

### B. 负形结合
- **方案**：在一个圆形或书籍形状中，利用负形切割出麦穗的轮廓。
- **推荐感**：简洁、专业、适合职业教育。

### C. 温暖曲线
- **方案**：采用柔和的金色曲线勾勒出抽象的麦穗，传达“温暖”和“阳光”的调性。
- **推荐色**：#f5c547 (金) + #f5ede1 (背景暖白)
