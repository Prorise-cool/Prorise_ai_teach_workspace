XiaoMai Design System
Confirm Dialog Component

Confirm Dialog 确认弹窗

用于在用户执行关键操作前进行二次确认。遵循 XiaoMai V4 实体化 (Solid UI) 规范：摒弃透明遮罩色块，采用纯色背景与高对比度边框，确保信息传达的极致清晰与专业感。

1. Danger (Delete / Remove)

永久删除该生成记录？

此操作不可恢复。删除后，系统将彻底清除该讲解视频及相关的伴学问答历史。

2. Warning (Leave / Reset)

确认离开当前测验？

您当前还有 3 道题未完成。离开后，已选择的答案将不会被保存。

3. Standard (Info / Action)

重置当前学习路径？

系统将根据您最近的表现重新规划路线。原有的进度和未解锁卡片将会被覆盖。

4. Success (Acknowledge)

课堂生成完毕！

您请求的《隐函数求导法》交互式课堂已准备就绪，可以随时开始学习。

永久删除该生成记录？

此操作不可恢复。删除后，系统将彻底清除该讲解视频及相关的伴学问答历史。
