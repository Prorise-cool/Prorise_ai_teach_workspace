XiaoMai Design System
Empty State Component

Empty State 空状态

当页面或组件无数据、出现错误、或需要引导用户操作时展示的占位模块。中心已严格预留 `120x120` 的画幅用于展示小麦的产品角色 (Mascot) 图像。

1. No Data (History/List)

暂无生成记录

你还没有使用小麦生成过任何课堂或视频，快去工作台试试吧。

2. No Results (Search/Filter)

未找到相关内容

换个关键词，或者减少一些筛选条件试试看。

3. Not Found (404)

页面走丢了

抱歉，你访问的页面不存在或已经被移除了。

4. Network Error (500)

网络连接异常

小麦暂时连不上服务器，请检查你的网络设置后重试。

5. Unauthorized (Auth Required)

需要登录

查看收藏夹与历史记录需要先登录小麦账号。

6. Full Page / Inline Container (Example)

区域内暂无内容
