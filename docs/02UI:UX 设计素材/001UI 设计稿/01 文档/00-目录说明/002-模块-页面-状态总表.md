# 模块、页面与状态总表

## 正式页面

| 模块 | 页面 / 路由 | 说明 | 数据宿主 |
|---|---|---|---|
| 首页与入口 | `/` | 双入口、推荐提示、认证入口 | RuoYi 认证 + 前端静态 |
| 视频学习 | `/video/input` | 文本 / 图片 / OCR / 风格选择 / 提交 | FastAPI |
| 视频学习 | `/video/:id/generating` | 统一等待体验 | FastAPI + SSE |
| 视频学习 | `/video/:id` | 视频结果与播放器 | FastAPI，部分收藏走 RuoYi |
| 课堂学习 | `/classroom/input` | 主题输入、风格入口 | FastAPI |
| 课堂学习 | `/classroom/:id/generating` | 课堂等待态 | FastAPI + SSE |
| 课堂学习 | `/classroom/:id` | 课堂结果、白板、多 Agent 扩展槽位 | FastAPI |
| 知识问答 | `/knowledge` | 问答、引用来源、文档上传 | FastAPI |
| 测验 | `/quiz/:sessionId` | 课后小测与即时反馈 | FastAPI |
| 学习路径 | `/path` | 目标、生成、保存、调整 | FastAPI |
| 学习沉淀 | `/history` | 历史记录与结果重开 | RuoYi |
| 个人域 | `/profile` | 资料查看与编辑 | RuoYi |
| 个人域 | `/favorites` | 收藏管理 | RuoYi |
| 个人域 | `/settings` | 设置入口壳层 | RuoYi |
| 学习中心 | 待定聚合页 | 聚合历史、收藏、问答、测验、路径 | RuoYi 为主，局部 FastAPI |

## 非路由交互
- 认证对话框：目标是首页弹出层，不是独立正式页面。
- 老师风格面板：输入页内部面板，不是独立正式页面。
- 统一等待壳层：视频 / 课堂共享，不是独立业务模块。
- 通用反馈状态：`loading`、`error-404`、`error-500`、`error-network`、`empty-state`、`confirm-dialog`、`toast`。

## 强约束
- 只有一套品牌视觉系统，主色为暖黄色 `#f5c547`。
- Agent 风格只能做头像、边框、标签、指示器等局部点缀。
- 当前 MVP 先以桌面端 `>= 1024px` 为主。
- 等待体验必须真实透明，不能用花哨伪进度代替真实阶段。

## 当前已知缺口

- `/classroom/:id`、`/profile`、`/history`、`/favorites`、`/settings` 仍需补规格。
