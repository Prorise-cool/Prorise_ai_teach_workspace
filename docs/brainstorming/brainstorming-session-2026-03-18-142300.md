## 5️⃣ 参考项目

### ManimToVideoClaw（现有项目）

**架构**：
```
用户请求 → scenext-forwarding (API网关)
                ↓
          storyboard (生成分镜脚本)
                ↓
          scenext (生成Manim代码 + 自动修复)
                ↓
          manimtovideo (渲染 + TTS + 上传OSS)
                ↓
          返回视频URL
```

**已有能力**：
- 完整的视频生成流水线
- 多 TTS 支持（豆包、百度、Spark、Kokoro）
- 自动代码修复（AST + LLM）
- 异步任务队列
- OSS 上传
