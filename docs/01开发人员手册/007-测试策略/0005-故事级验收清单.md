# 故事级验收清单

> **状态**: 已初始化
> **负责人**: Dev / Reviewer / QA
> **最后更新**: 2026-03-29

---

## Ready 清单

- [ ] Story 目标、范围、AC 与依赖明确
- [ ] Story Type 已标注
- [ ] Contract Asset Path / Mock Asset Path / Persistence Impact 已标注
- [ ] 关键错误态、空态、降级态已写入 Story
- [ ] Frontend Story 已有批准的高保真视觉稿
- [ ] Backend / Contract Story 已有稳定 schema、错误码、状态枚举与示例

## 开发自检清单

- [ ] 改动范围与当前 Story / Issue 对齐
- [ ] 代码、契约、文档、测试或模板交付与 Story 类型匹配
- [ ] 至少完成一轮本地验证，并能给出命令或证据
- [ ] 不存在“只有页面壳”“只有接口壳”“只有 README”的伪完成状态
- [ ] 未突破 FastAPI / RuoYi / Redis / COS / Provider 边界

## 审查清单

- [ ] AC 已逐条覆盖
- [ ] 测试、日志、截图或文档证据充分
- [ ] mock、真实接口与文档口径一致
- [ ] 索引、规范引用与模板已按需更新
- [ ] Blocker / Major 问题已关闭

## 合并 / 发布清单

- [ ] PR 已关联 Story / Epic / Issue
- [ ] PR 描述填写验收、自测、风险与回滚信息
- [ ] 合并前门禁满足
- [ ] 若涉及发布，则发布前门禁满足
