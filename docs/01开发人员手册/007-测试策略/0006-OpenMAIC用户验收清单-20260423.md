## 如果出问题 — 诊断指引

| 症状 | 检查点 |
|------|--------|
| 点"生成课堂"一直停在 5% | 终端看 Dramatiq worker 是否报错；`curl http://localhost:8090/api/v1/openmaic/bootstrap` 带 Bearer 看是否 200 |
| 跳转后课堂页空白 | DevTools Network → 看 `GET /api/v1/openmaic/classroom/{id}` 响应体的 `status` 字段是否是 `ready` |
| 登录后一直跳回登录页 | RuoYi 8080 挂了，`curl http://localhost:8080/` 应该 200 |
| 所有 `/api/*` 401 | `xiaomai-auth-session` localStorage 过期，重新登录即可 |
