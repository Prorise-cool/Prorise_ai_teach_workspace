# Epic 9 学习中心与个人域前端点击验收清单（/learning /history /favorites /profile /settings）

> 事实源以 [`../../../_bmad-output/planning-artifacts/epics/26-epic-9.md`](../../../_bmad-output/planning-artifacts/epics/26-epic-9.md) 为准。本文只给前端点击验收口径。

## 本次验收要确认什么

1. 学习中心域：`/learning` 聚合页可用，并能进入 `/history` 与 `/favorites`。
2. 历史/收藏：分页列表可用，收藏/取消收藏/删除操作有反馈，收藏页支持文件夹与移动。
3. 个人资料：`/profile` 可编辑并保存（本地持久化），头像上传即时预览可用。
4. 设置页：`/settings` 的主题/语言/通知开关可用；账号安全项（手机号/密码/会话）不再是 “coming soon toast”，而是可交互的真实弹层与动作。

## 点击验收清单

### A. 学习中心聚合页（/learning）

1. 打开 `/learning`，确认聚合模块正常渲染（继续学习/收藏/错题本/依据溯源/学习路径/小测与推荐等）。
2. 点击页面内的“历史记录”入口，确认进入 `/history`（带或不带 `resultType` 均可）。
3. 点击页面内的“我的收藏”入口，确认进入 `/favorites`。

### B. 历史记录（/history）

1. 在 `/history` 点击任意记录卡片标题/CTA，确认不会报错（mock 模式下允许只演示打开与跳转，不要求真实详情页存在）。
2. 对任意记录执行“收藏/取消收藏”（星标）操作，确认状态有即时反馈。
3. 对任意记录执行“删除”操作，确认有成功/失败反馈且列表状态更新。

### C. 收藏页（/favorites）

1. 在 `/favorites` 点击“新建文件夹”，输入名称并确认，看到新文件夹出现在列表中（本地持久化）。
2. 对任意收藏记录点击“移动”，选择目标文件夹并确认，看到记录显示当前所在文件夹信息。
3. 点击“取消收藏”，确认该条记录从收藏列表中移除或状态更新。

### D. 个人资料（/profile）

1. 打开 `/profile`，修改“用户昵称”后点击“保存个人资料”，确认出现保存成功反馈。
2. 刷新页面，确认昵称与扩展信息仍被保留（本地持久化）。
3. 点击头像区域“上传新图片”，选择一张图片，确认头像即时预览更新。
4. 点击“取消更改”，确认草稿回退到最近一次保存的状态。

### E. 设置页（/settings）

1. 打开 `/settings`，切换“主题模式”为浅色/深色/跟随系统，确认 UI 主题随之变化。
2. 切换“界面语言”为 English，确认页面关键文案发生变化（i18n 生效）。
3. 切换“站内通知提醒”开关，确认出现成功反馈。
4. 点击“更换手机号”，确认弹层出现；输入 `13912345678` 并保存，确认绑定手机号显示更新为 `139 **** 5678`。
5. 点击“修改密码”，确认弹层出现；输入旧密码/新密码/确认密码并保存，确认有成功反馈并关闭弹层。
6. 点击“查看登录设备”，确认弹层出现；点击“退出当前会话”，确认跳转到 `/login` 并清理本地登录态。
7. 重新登录后返回 `/settings`，点击“清理缓存”，确认有成功反馈且本地缓存项被清空（主题/通知/收藏文件夹/手机号等）。

## 预期结果

- Epic 9 的五个正式路由页均可在 mock 模式下完成核心交互与状态闭环。
- 设置页三项安全入口（手机号/密码/会话）均为真实可用交互，不再只 toast “coming soon”。
- 所有新增 UI 不写死颜色，遵守全局 token 与亮/暗色语义一致性。
