## API 自测入口

- 结果详情：`GET /api/v1/video/tasks/{task_id}/result`
  - 预期返回 `VideoResultDetail`，成功态含 `result`，失败态含 `failure`。
- 公开发布：`POST /api/v1/video/tasks/{task_id}/publish`
  - 预期仅任务创建者可操作，成功后返回 `PublishedVideoCard`。
- 取消公开：`DELETE /api/v1/video/tasks/{task_id}/publish`
  - 预期幂等撤回公开状态。
- 公开列表：`GET /api/v1/video/published?page=1&pageSize=12`
  - 预期按 `publishedAt DESC` 返回分页卡片。

## 运行态与降级说明

- 本地 / 测试环境默认使用 `LocalAssetStore` 与 `stub-llm` / `stub-tts`，便于单测和集成测试稳定运行。
- `DockerSandboxExecutor` 在宿主机无 Docker 时自动回退到 `LocalSandboxExecutor`。
- FFmpeg 不可用或渲染产物为测试占位文件时，合成阶段会走 placeholder fallback。
- 视频主结果与 artifact graph 通过本地资产目录模拟对象存储引用；长期元数据通过 RuoYi 防腐层接口回写到 `xm_video_task`、`xm_user_work`、`xm_session_artifact`。
