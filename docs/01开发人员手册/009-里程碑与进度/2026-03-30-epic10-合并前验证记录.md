### 2. RuoYi 模块测试执行链路

根因：

- 根 `pom.xml` 的 `maven-surefire-plugin` 使用 `groups=${profiles.active}`
- 默认环境是 `dev`
- `ruoyi-xiaomai` 测试类原先只有 `@Tag("local")`
- 所以之前会出现测试源码编译通过，但执行结果是 `Tests run: 0`

处理：

- 为 `ruoyi-xiaomai` 下可安全执行的测试类补充 `@Tag("dev")`

执行命令：

```bash
mvn -pl ruoyi-modules/ruoyi-xiaomai -am -DskipTests=false test
```

结果：

- `Tests run: 17, Failures: 0, Errors: 0, Skipped: 0`
- `BUILD SUCCESS`
