### 小麦项目使用边界

1. Redis 仅承载运行时状态、认证态和事件缓存。
2. 以下数据不得仅保存在 Redis：
   - 学习记录
   - 收藏
   - 视频任务元数据
   - 课堂历史
   - 会话摘要
3. 小麦运行时 Key 统一遵循：
   - `xm_task:{task_id}`
   - `xm_task_events:{task_id}`
   - `xm_video_runtime:{task_id}`
   - `xm_classroom_runtime:{session_id}`
   - `xm_provider_health:{provider}`

## Docker Compose

参考文件：`packages/RuoYi-Vue-Plus-5.X/script/docker/docker-compose.yml`

### 最小启动命令

```bash
cd packages/RuoYi-Vue-Plus-5.X/script/docker
docker compose up -d mysql redis minio
```

### 注意事项

1. 该 Compose 文件使用了大量宿主机绝对路径挂载，例如 `/docker/mysql/data/`、`/docker/redis/conf/`。
2. 该 Compose 文件使用 `network_mode: "host"`，在 macOS / Windows 上可能需要改写为普通端口映射后再使用。
3. 团队若希望把它作为长期本地开发方案，建议后续复制为项目私有的精简版 Compose。

## Story 1.1 最小中间件要求
