## 参考项目

| 项目名称 | 仓库位置 | 参考内容 | 许可证 | 说明 |
|----------|----------|----------|--------|------|
| **OpenMAIC** | `references/openmaic/` | LangGraph 多 Agent 编排、课堂生成流水线、动作执行引擎、LLM/TTS 服务商抽象、SSE 流式传输 | AGPL-3.0 | 开源多智能体互动课堂平台，Next.js + LangGraph 架构。课堂服务模块的核心复用来源，需将 Next.js API 移植为 FastAPI |
| **manim-to-video-claw** | `references/manim-to-video-claw/` | 分镜生成（storyboard）、Manim 代码生成与自修复（scenext）、渲染合成（manimtovideo）、多 TTS 服务商支持、API 网关 | 商业自有 | AI 驱动的 Manim 视频生成流水线，包含完整的题目理解→分镜→代码→渲染→合成流程。视频生成服务模块的核心复用来源 |
| **RuoYi-Vue-Plus-5.X** | `packages/RuoYi-Vue-Plus-5.X/` | ToB 用户管理、RBAC 权限控制、JWT Token 认证（sa-token）、系统配置管理 | MIT | Java Spring Boot 管理后台框架。已部署为 ToB 后端服务（:8080），FastAPI 通过共享 Redis 验证 JWT，无需 HTTP 调用 |
| **ruoyi-plus-soybean-master** | `packages/ruoyi-plus-soybean-master/` | ToB 管理端前端界面、Vue 3 + Naive UI 组件模式 | MIT | Vue 3 管理前端，配合 RuoYi-Vue-Plus 使用。已有项目，赛后扩展 |

### OpenMAIC 关键目录

| 目录 | 用途 | 复用方式 |
|------|------|----------|
| `openmaic/lib/orchestration/` | LangGraph 工作流编排（课堂生成核心） | 逻辑移植到 FastAPI |
| `openmaic/lib/generation/` | 场景内容/动作/幻灯片生成 | 逻辑移植到 FastAPI |
| `openmaic/lib/pbl/` | 项目制学习 + 交互式测验 | 逻辑移植到 FastAPI |
| `openmaic/app/api/` | ~18 个 API 端点（课堂、聊天、TTS、搜索等） | 接口设计参考 |
| `openmaic/app/api/generate/` | 场景生成流水线 API（图片/TTS/视频/大纲） | 核心复用 |
| `openmaic/app/api/chat/route.ts` | 多 Agent 讨论 SSE 流式传输 | 移植为 FastAPI SSE |

### manim-to-video-claw 关键目录

| 目录 | 用途 | 复用方式 |
|------|------|----------|
| `manim-to-video-claw/storyboard/` | 分镜生成服务（FastAPI） | 合并入统一 FastAPI |
| `manim-to-video-claw/scenext/` | Manim 代码生成 + 自动修复 | 合并入统一 FastAPI |
| `manim-to-video-claw/scenext/auto_fix_code/` | AI/AST/渲染错误 三级修复 | 直接复用 |

| `manim-to-video-claw/manimtovideo/manimtovideo/` | 多 TTS 服务商（豆包/百度/Spark/Kokoro） | 迁移为抽象工厂 |
| `manim-to-video-claw/scenext-forwarding/` | API 网关 | 参考路由设计 |
