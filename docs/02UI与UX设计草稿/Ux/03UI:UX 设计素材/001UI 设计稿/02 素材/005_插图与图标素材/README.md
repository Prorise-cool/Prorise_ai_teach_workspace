## C. 装饰元素与背景 (Patterns & Textures)

### 1. 几何纹理背景
- **Hero Patterns**: [heropatterns.com](https://www.heropatterns.com/)
    - 推荐模式：`Jigsaw` (拼图)、`Topography` (地形图)、`Plus` (加号)。
    - 用法：在卡片背景或页面分界处使用 `#f5c547` 且不透明度为 0.05 - 0.1 的 SVG 背景。

### 2. "小麦"品牌元素 (Wheat Grains)
- **SVG Repo (Wheat Grains)**: [svgrepo.com](https://www.svgrepo.com/svg/196770/wheat-grain)
- **UXWing (Grain Wheat)**: [uxwing.com/grain-wheat-icon/](https://uxwing.com/grain-wheat-icon/)
    - 授权：CC0 (无需署名)。
    - 场景：在页面底角、按钮点缀或加载动画中作为品牌符号。

### 3. 毛玻璃效果 (Glassmorphism)
- **CSS.glass**: [css.glass](https://css.glass/)
    - 建议：在暖金色背景上叠加高斯模糊 (Blur: 10-20px) 的白色卡片，营造温暖且现代的专业感。

## 品牌视觉调配建议 (Brand Sync)

1. **插图配色**: 所有 SVG 插图的 Accent Color 统一设为 `#f5c547`。
2. **图标选择**: 优先使用 **Phosphor Icons (Regular/Duotone)**。Duotone 模式下的次要色建议设为 `#f5c547` 的 20% 透明度。
3. **装饰元素**: 在 AI 对话框背景使用极淡的 `Topography` SVG 纹理。
