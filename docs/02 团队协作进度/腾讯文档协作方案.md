## 一、方案概述

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **在线协作** | 所有文档必须在腾讯文档在线编辑，禁止在群里传本地文件 |
| **实时同步** | 文档修改后立即同步，其他成员可看到最新版本 |
| **职责明确** | 每个文档有且仅有1个主要负责人 |
| **命名规范** | 所有文档按统一格式命名，便于识别 |

### 1.2 分工概览

| 成员 | 角色 | 腾讯文档职责 |
|------|------|-------------|
| 彭润润 | 全流程研发 | 技术架构文档、产品链接、里程碑进度 |
| 黄坍瑶 | PPT创意 | 竞品分析表、品牌视觉规范、PPT初稿 |
| 朱茗熙 | PPT调研 | 用户调研记录、需求文档、演示脚本 |
| 古佳玉 | UI设计 | 设计规范、设计稿链接、素材包 |
| 余晓婷 | 视频剪辑 | 视频脚本、素材清单、成品链接 |
| 包芷凝 | 文档统筹 | 方案说明书（统筹）、答辩稿、周报汇总 |
| 林楷音 | 内容策展 | 赛项数据包、案例库、进度跟踪 |
