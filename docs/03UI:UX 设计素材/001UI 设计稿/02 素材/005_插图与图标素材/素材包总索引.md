## 二、图标库选型说明

### 为什么选择 Phosphor Icons？

| 评估维度 | Phosphor Icons | 匹配度 |
|----------|---------------|--------|
| **风格** | 圆润、友好、线条温和 | ⭐⭐⭐⭐⭐ 与小麦 Logo 风格高度统一 |
| **图标数量** | 9,000+ | 覆盖所有页面需求 |
| **变体** | Regular / Fill / Duotone / Thin / Light / Bold | 支持不同状态 |
| **React 集成** | `@phosphor-icons/react` | 原生 React 组件 |
| **License** | MIT | 商用无限制 |
| **Tree-shaking** | 支持 | 按需引入，不影响包体积 |

### React 项目中使用方式

```bash
npm install @phosphor-icons/react
```

```tsx
import { House, Play, GraduationCap, Sparkle } from '@phosphor-icons/react';

// Regular (默认线性)
<House size={24} />

// Fill (填充/选中态)
<House size={24} weight="fill" />

// Duotone (双色装饰)
<House size={24} weight="duotone" />

// 带品牌色
<Sparkle size={24} color="#f5c547" weight="duotone" />
```

> **注意**：实际开发中推荐通过 `@phosphor-icons/react` npm 包按需引入，本素材包中的 SVG 文件用于设计稿参考和非 React 场景。

## 六、插图素材获取指南

对于复杂场景插图（人物、场景画），需要从以下平台手动下载 SVG：

### 推荐平台

| 平台 | 授权 | 特点 | 品牌色适配 |
|------|------|------|-----------|
| **Storyset** | 免费 (需署名) | Amico 风格最温暖，5 种统一风格 | 支持在线换色 `#f5c547` |
| **unDraw** | MIT (完全免费) | 极简，契合 Shadcn 审美 | 支持在线换色 |
| **LottieFiles** | 部分免费 | JSON 动画，适合等待状态 | 部分可换色 |

### 下载步骤

1. 打开平台链接（各子目录 README.md 中有具体 URL）
2. 将 Accent Color / 主色设为 `#f5c547`
3. 选择 Amico 风格（Storyset）以保持统一
4. 导出 SVG 格式
5. 保存到对应子目录

各子目录的 `README.md` 包含针对该场景的精确下载链接和推荐搜索关键词。

## 七、开发集成建议

### NPM 安装

```bash
npm install @phosphor-icons/react
```

### 全局图标组件封装

```tsx
// components/ui/icon.tsx
import { IconProps } from '@phosphor-icons/react';

interface XiaoMaiIconProps extends IconProps {
  /** 是否使用品牌色 */
  branded?: boolean;
}

export function Icon({ branded, color, ...props }: XiaoMaiIconProps) {
  return (
    <props.icon
      color={branded ? '#f5c547' : color}
      {...props}
    />
  );
}
```

### CSS 变量配合

```css
:root {
  --icon-color-default: var(--text-primary);
  --icon-color-brand: #f5c547;
  --icon-color-muted: var(--text-secondary);
  --icon-size-sm: 16px;
  --icon-size-md: 20px;
  --icon-size-lg: 24px;
  --icon-size-xl: 32px;
}
```

## 八、注意事项

1. **统一性**：全项目只使用 Phosphor Icons 一个图标库，禁止混用其他图标库
2. **权重一致**：默认使用 `regular` 权重，选中/激活态使用 `fill`，装饰性使用 `duotone`
3. **尺寸规范**：导航图标 20px，内容图标 16-24px，装饰图标 32px+
4. **颜色规范**：图标默认继承文字颜色 (`currentColor`)，品牌强调用 `#f5c547`
5. **无障碍**：所有交互图标必须配 `aria-label`，装饰图标用 `aria-hidden="true"`
