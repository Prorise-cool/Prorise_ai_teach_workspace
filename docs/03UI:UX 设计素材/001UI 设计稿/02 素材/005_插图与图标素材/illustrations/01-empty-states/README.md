# 空状态插图下载指南

以下插图需要从 Storyset / unDraw 手动下载，设置品牌色 `#f5c547` 后导出 SVG/PNG。

## 推荐插图

### Storyset (免费，需署名)
> 进入链接后，点击右侧调色盘将 Accent Color 设为 `#f5c547`，然后点击 Download

| 场景 | 下载链接 | 文件名 |
|------|----------|--------|
| 无数据 | https://storyset.com/illustration/no-data/amico | `no-data.svg` |
| 搜索无结果 | https://storyset.com/illustration/searching/amico | `search-empty.svg` |
| 空收藏夹 | https://storyset.com/illustration/void/amico | `empty-collection.svg` |
| 无历史记录 | https://storyset.com/illustration/blank-canvas/amico | `no-history.svg` |

### unDraw (MIT 免费)
> 进入 https://undraw.co/illustrations，顶部颜色选择器设为 `#f5c547`，搜索关键词下载

| 搜索关键词 | 推荐场景 | 文件名 |
|-----------|----------|--------|
| `void` | 通用空状态 | `undraw-void.svg` |
| `empty` | 空列表 | `undraw-empty.svg` |
| `no data` | 无数据 | `undraw-no-data.svg` |
