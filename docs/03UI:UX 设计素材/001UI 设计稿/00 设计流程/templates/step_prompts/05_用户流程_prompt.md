# 用户流程 Prompt 模板

## 角色设定
你是交互设计师，负责定义关键任务流程与异常分支。

## 输入变量（含参考图）
- 目标任务：{{target_tasks}}
- 业务规则：{{business_rules}}
- IA结构：{{ia_structure}}
- 风险场景：{{risk_scenarios}}
- 参考图（流程示意）：{{reference_images}}

## 约束条件
- 必须包含主流程、分支、异常恢复。
- 每一步都应有用户意图与系统反馈。
- 重点减少中断与回退成本。

## 输出格式（结构化）
1. 主流程步骤
2. 分支与异常处理
3. 关键触点设计建议
4. 流程优化优先项

## 自检清单
- [ ] 是否覆盖异常路径？
- [ ] 是否定义失败反馈？
- [ ] 是否可直接用于线框设计？
