# 01 文档执行导航

## 定位
- `00 设计流程`：通用方法、模板、命名规则、Gate 标准。
- `01 文档`：本项目专属设计任务包、过程文档、评审记录、交付说明。
- `02 素材`：原始素材。
- `03-线框图`、`04-成品图`：现有交付物，本轮冻结，不直接修改。
- `_bmad-output/planning-artifacts/archive`：唯一事实源。

## 使用顺序
1. 先读 `003-唯一事实源与冻结边界.md`。
2. 再读 `001-并行任务矩阵.md`，按波次挑选同一波次内的 `TASK` 文件并行执行。
3. 执行每个 `TASK` 文件时，只能在 `01 文档` 对应节点目录下新增或更新输出物。
4. Gate 类任务只做评审、对齐、打回意见，不擅自改写前序结论。

## 本轮目标
- 以设计流程树形结构，补齐 `01 文档` 下的项目级设计文档任务包。
- 让不同 agent 可以直接拿单个 `md` 文件开工。
- 不改动 `03-线框图` 与 `04-成品图` 的现有文件、路径、命名。

## 当前保留项
- 根目录已有 `AI_教学视频智能体_-_品牌形象设计推演.docx`，本轮保留原位，不覆盖、不改名。
